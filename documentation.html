Jag har med hjälp av HTML och CSS skapat denna websidan som är ett CV om mig, mina kurser och intressen.

Eftersom detta projektet är första gången jag har skrivit kod så tog det lång tid för mig att komma igång, speciellt med CSS. Men när jag väl lärde mig grunderna var det kul och enklare än jag trodde.

Jag hade problem med bakgrundsbilden eftersom jag trodde att man kunde lägga in nedladdade filer i CSS och att jag hade gjort fel när jag lade in bild-adress istället. Så jag lade ner väldigt mycket tid på att försöka hitta ett fel som inte existerade. jag laddade ner i samma fil utan behövde länka bildadress för att kunna använda de.

När det var något jag inte förstod kollade jag igenom gamla freecodecamp lektioner eller kollade tutorials och lektioner på Youtube. Två sidor som hjälpte mig mycket var Stack Overflow och W3Schools
